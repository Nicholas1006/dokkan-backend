Dokkan Asset Downloader v2.2.2

Note: Asset downloader will symlink dokkan assets to mark which files are new. This requires that the program run as admin.

Note: This is not open source software. There is no warranty for using this software. This program was created during the UniDokkan era and distributed to those in the community supporting open and sharing of knowledge for Dokkan and the modding scene. An open source recommended good start for learning the Dokkan Game API is https://github.com/pawelorzech/dokkan-battle-bot

Commands:
	- 1 - Run downloader
	- 2 - Enable downloading of Japan Dokkan assets (requires additional settings)
	- 3 - Enable downloading of Global Dokkan assets
	- 4 - Download and unpack Dokkan assets in memory (disable on low memory systems)
	- 5 - Delete CPK files after unpacking is complete
	- 6 - Do not unpack CPK files
	- 7 - Folder location in which to download assets (recommended to be an unused folder)
	- 8 - [JP] Mobile device name for downloading Japan Dokkan files (see below)
	- 9 - [JP] Mobile device model for downloading Japan Dokkan files (see below)
	- 0 - [JP] Mobile device os version for downloading Japan Dokkan files (see below)
	- A - [JP] Single use access token generated by a real device (see below)
	- B - [JP] Single use auth secret generated by a real device (see below)
	- C - Japan game version obtained from game library (see below)
	- D - Start timestamp to download asset files (zero for all files; updated automatically)
	- E - Start timestamp to download database file (zero for all files; updated automatically)
	- F - Japan database password for decrypting database
	- G - Same as C but for Global Dokkan
	- H - Same as D but for Global Dokkan
	- I - Same as E but for Global Dokkan
	- J - Same as F but for Global Dokkan
	- K - [GLB] Global game version files to download, based on country identifier (separated by commas)
	- L - Folder to symlink new files each time download is run (WARNING! WILL DELETE FOLDER ON RUN)
	
Required Settings:
You MUST provide the APK version and database passwords for the version of the game you are attempting to download. These can be found inside the game library or by asking someone in the community.	

Japan Authentication:
Since the enforcement of SafetyNet and Device Attestation, to be able to download Japan Dokkan assets, you need to have a rooted/jailbroken device that can run Dokkan. With such a device, using network packet capturing, you can obtain the Dokkan authentication tokens used to access the game. These tokens are temporary and last 1 hour. 

Global Account:
On first global download use, the program will attempt to create a new global account. You will be asked to complete a captcha. Since the program will not attempt the tutorial, this account is temporary and will eventually be deleted by the game. Though not required, you may provide your own account details for a more established Dokkan account by editing "account_global.json". It is recommended to not use your main Dokkan account if you do this.
